# ChatApp
### Made with React and the Pusher ChatKit SDK.

## How to use:

- Clone this repository

`git clone https://github.com/sherialeliasjawed/chatapp.git`

- Install dependencies

`yarn`

- Start yarn

`yarn start`

Enjoy!
